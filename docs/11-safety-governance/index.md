# 可信、安全与 AI 治理

本目录整理可信 AI、安全风险、隐私保护、可解释性和治理问题。它面向研究和系统实现，而不是合规口号。

## 建议主题

- 对抗样本、数据投毒、后门攻击、模型窃取
- Prompt Injection、越权工具调用、敏感信息泄露
- 幻觉、事实性、偏见、公平性、校准性
- 隐私保护、成员推断、差分隐私、联邦学习
- 可解释性、可审计性、可追踪性和模型卡
- Red Teaming、安全评测、数据许可和模型许可

## 关键问题

- 威胁模型是什么，攻击者能力如何假设
- 模型输出是否需要事实校验、引用来源和不确定性估计
- 数据、模型和代码许可是否支持研究复现和公开发布
- 安全策略是否能被实验评测，而不是只停留在规则描述
- 风险控制如何影响模型能力、可用性和研究结论
