# 训练、微调与对齐

本目录沉淀模型训练、预训练、微调、参数高效微调和对齐方法。

## 建议主题

- 预训练、继续预训练、监督微调
- LoRA、QLoRA、Adapter、Prompt Tuning
- RLHF、DPO、RLAIF、偏好优化
- 分布式训练、混合精度、梯度检查点
- Checkpoint、实验管理、可复现训练
- 训练稳定性、过拟合、灾难性遗忘

## 关键问题

- 训练目标和数据分布是否匹配
- 评测集是否能反映真实目标
- 训练成本和收益是否可解释
- 微调后的模型是否破坏基础能力
- 对齐方法是否引入新的偏差或安全风险
