# AI Knowledge Graph

这是一个开放 AI 知识库的入口，用于沉淀 AI 基础理论、模型与任务、数据工程、训练微调、推理应用、AI 系统、评测优化、安全治理和 AI 可读知识组织方式。

## 目标

- 为学习者和实践者提供体系化学习资料。
- 为工程实践者提供从模型到应用、从软件到基础设施的结构化 AI 资料。
- 为 AI 检索、问答、知识图谱和 skills 提供结构化知识源。

## 写作原则

- 每篇文档必须包含 front matter 元数据。
- 重要结论需要标明来源、适用范围和验证状态。
- 测试数据、经验总结和方案比较需要标注来源、上下文和 owner。
- 临时讨论不要直接进入知识库，先沉淀成可复用知识点。

## 推荐阅读路径

1. 从 `知识地图` 理解整体结构。
2. 入门读者先阅读 `AI 基础理论`、`模型与任务`、`数据与数据工程`。
3. 应用开发者重点阅读 `推理与应用构建`、`Prompt、RAG 与 Agents`、`AI 软件栈`。
4. 系统工程读者重点阅读 `AI 计算与基础设施`、`系统、集群与 MLOps`、`评测、Benchmark 与优化`。
5. AI 检索系统优先读取 Markdown 源文件和元数据。
