# AI Knowledge Graph

这是一个面向 AI 方向研究生新生、研究者和 AI agent 的开放知识库。它以学术训练和技术研究为主线，沉淀数学基础、模型架构、数据集与基准、训练算法、推理系统、计算架构、可信 AI、论文复现和研究工作流。

## 目标

- 为课题组新生提供一条从基础课到论文阅读、实验复现、系统实现的学习路径。
- 为研究者整理模型方法、系统方法、评测方法和实验记录。
- 为 AI 检索、问答、知识图谱和 skills 提供可引用、可追溯的结构化知识源。

## 写作原则

- 每篇文档必须包含 front matter 元数据。
- 重要结论需要标明论文、教材、代码仓库或实验记录来源。
- 实验结论需要保留任务设定、数据集、指标、硬件环境、软件版本和复现状态。
- 技术比较需要说明前提假设、适用范围和反例。
- 临时讨论不要直接进入知识库，先沉淀成可复用的知识点、论文笔记或实验记录。

## 推荐阅读路径

1. 从 `知识地图` 理解整体结构。
2. 新生先阅读 `数学与机器学习基础`、`模型架构与任务范式`、`数据集、标注与基准`。
3. 做模型方向的同学重点阅读 `训练算法、微调与对齐`、`评测、复现与性能分析`、`可信、安全与 AI 治理`。
4. 做系统方向的同学重点阅读 `推理算法与高效服务`、`框架、编译器与运行时`、`AI 计算架构与硬件基础`、`分布式系统与实验平台`。
5. 做论文复现和组内知识沉淀时，优先使用 `论文复现、研究案例与决策记录`、`知识组织与研究工作流` 和模板。
