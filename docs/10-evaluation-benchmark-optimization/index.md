# 评测、复现与性能分析

本目录关注 AI 研究中的评测方法、实验复现、性能分析和基准测试。它是判断论文结论是否可靠、系统优化是否有效的核心环节。

## 建议主题

- 模型能力评测、任务评测、人工评估、偏好评估
- 幻觉、忠实度、安全性、鲁棒性、校准性评测
- Benchmark 设计、数据污染、统计显著性和置信区间
- 论文复现协议、消融实验、对照实验
- 训练性能、推理性能、通信性能和端到端性能
- Profiling、Roofline、Trace、火焰图和瓶颈定位
- 能效、稳定性、可扩展性和资源效率指标

## 关键问题

- 指标是否真正对应研究问题
- 实验差异来自算法、数据、实现还是硬件环境
- 性能优化是否牺牲模型质量、鲁棒性或可复现性
