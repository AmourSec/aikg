# Prompt、RAG 与 Agents

本目录沉淀提示词工程、上下文工程、检索增强生成、工具调用和智能体系统知识。

## 建议主题

- Zero-shot、Few-shot、Chain-of-Thought、ReAct
- Prompt Chaining、Context Engineering
- Embedding、向量数据库、Hybrid Search、Rerank
- RAG 数据切分、召回、引用、评测
- Tool Calling、Function Calling、MCP
- Agent 架构、记忆、规划、工作流和边界控制

## 关键问题

- 任务应该靠提示词、检索、微调还是工具解决
- 知识来源是否可靠、可追溯、可更新
- Agent 是否有清晰权限、状态和失败处理
- RAG 是否能评测召回质量和答案忠实度
- 提示词和工具接口是否适合沉淀为 AI skills
