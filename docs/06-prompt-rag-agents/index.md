# 检索增强、工具使用与智能体

本目录关注大模型与外部知识、工具和环境交互的研究问题，包括上下文学习、检索增强生成、工具使用、规划和智能体评测。

## 建议主题

- Zero-shot、Few-shot、Chain-of-Thought、ReAct
- In-context Learning、Prompt Chaining、Context Engineering
- Embedding、向量数据库、Hybrid Search、Rerank
- RAG 切分、召回、引用、忠实度和归因评测
- Tool Calling、Function Calling、MCP、程序合成
- Agent 规划、记忆、反思、多智能体协作和环境反馈
- WebArena、AgentBench、ToolBench 等评测范式

## 关键问题

- 上下文学习、检索、微调和工具使用分别适合什么问题
- 检索结果是否可追溯，生成答案是否忠实于来源
- Agent 的状态、工具权限和失败恢复如何建模
- 评测是否能覆盖长程任务、部分可观测环境和工具错误
- 可复用提示词、工具接口和实验协议如何沉淀为 AI-readable skills
