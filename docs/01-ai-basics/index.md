# 数学与机器学习基础

本目录面向入门研究生，沉淀理解现代 AI 论文所需的数学、统计学习和深度学习基础。

## 建议主题

- 线性代数、概率统计、信息论、最优化
- 统计学习理论、泛化、偏差-方差、正则化
- 神经网络基础、反向传播、初始化、归一化
- 表示学习、序列建模、注意力机制
- 论文阅读所需的常用符号、实验术语和指标

## 入门要求

- 能读懂基本矩阵运算、概率分布、梯度下降和交叉熵。
- 能把一篇模型论文拆解为问题定义、模型假设、损失函数、训练过程和评测指标。
