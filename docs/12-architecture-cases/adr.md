# 研究决策记录

研究决策记录用于保留重要技术选择、实验设计和系统实现取舍，帮助后来者理解“当时为什么这样做”。

## 何时记录

- 选择某篇论文作为复现基线或对照方法。
- 改变数据集、评测指标、训练策略或硬件配置。
- 对性能、能效、稳定性、可复现性有长期影响的实现选择。
- 不同方案各有明显代价，需要保留判断依据、假设和后果。
- 未来读者或 AI agent 需要理解实验脉络和决策依据。
