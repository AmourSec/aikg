# 数据与数据工程

本目录沉淀 AI 数据集、数据处理、标注、清洗、质量控制和数据管道知识。

## 建议主题

- 数据采集、清洗、去重、脱敏
- 数据标注、主动学习、弱监督
- Dataset、DataLoader、数据版本管理
- Tokenization、特征工程、数据增强
- Synthetic Data、数据合成、数据蒸馏
- 数据质量、数据泄漏、分布漂移

## 关键问题

- 数据是否覆盖目标场景
- 标注质量如何验证
- 训练、验证、测试集如何切分
- 数据处理流程是否可复现
- 数据能否被模型、评测和 AI 检索系统共同使用
