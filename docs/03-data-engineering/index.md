# 数据集、标注与基准

本目录关注 AI 研究中的数据集构建、标注协议、数据质量控制和 benchmark 设计。数据不仅是工程资源，也是实验结论能否成立的前提。

## 建议主题

- 数据采集、清洗、去重、脱敏
- 标注协议、一致性检验、主动学习、弱监督
- Dataset、DataLoader、数据版本和数据卡
- Tokenization、特征工程、数据增强
- Synthetic Data、数据合成、数据蒸馏和自训练
- 数据质量、数据泄漏、分布偏移和污染检测
- 训练集、验证集、测试集和公开 benchmark 的边界

## 关键问题

- 数据是否支持论文中的研究问题
- 标注协议是否可复核，标注一致性如何度量
- 训练、验证、测试集如何切分，是否存在泄漏
- 数据处理流程是否可复现、可追踪、可引用
- 数据集能否支撑模型训练、评测和后续知识检索
