## 建议主题

- PyTorch、JAX、ONNX
- CUDA、ROCm、Triton、CUTLASS、cuDNN、NCCL
- Kernel、算子融合、内存规划、自动调优
- TorchInductor、XLA、TVM、MLIR、TensorRT
- Runtime、调度系统、异步执行和图捕获
- 分布式训练、分布式推理和通信库
