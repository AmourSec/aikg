# AI Skill Samples

This directory contains small, task-oriented skill samples for AI agents.

The documentation in `docs/` explains concepts for humans and AI retrieval. The files in `skills/` are different: they describe repeatable workflows that an AI agent can follow when doing a concrete task.

Current sample:

- `npu-arch-capability-check/SKILL.md`: a minimal workflow for checking Ascend NPU model, SocVersion, NpuArch, architecture-specific branches, and evidence gaps.

These samples are intentionally compact. New skills should be added only when a workflow is repeated often enough to justify a reusable checklist and output template.
